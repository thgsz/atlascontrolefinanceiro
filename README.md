## ✨ Sobre o projeto

O **Atlas** nasceu de uma necessidade real:

> “Nenhum aplicativo de finanças era exatamente do jeito que eu queria.”

Então resolvi criar o meu próprio.

O Atlas é um aplicativo de **gestão financeira pessoal**, com foco em:

- 💰 controle de receitas e despesas
- 📊 visão clara da vida financeira
- 🧠 insights inteligentes
- 🔒 privacidade rápida estilo app bancário
- ⚡ experiência rápida e fluida
- 🎨 design premium inspirado em fintechs

Tudo isso em uma interface moderna, bonita e intuitiva.

---

## 🚀 Funcionalidades

### 💰 Controle Financeiro

- Adicionar receitas e despesas
- Balanço automático do mês
- Saldo disponível calculado automaticamente
- Histórico de transações
- Categorias personalizadas
- Controle mensal por data

---

### 📊 Dashboard Inteligente

- Resumo financeiro mensal
- Receitas vs despesas
- Gráfico **Raio-X Financeiro**
- Indicador de saúde financeira
- Visualização limpa e rápida

---

### 🧠 Atlas Insights

Análises automáticas sobre hábitos financeiros:

- maiores gastos do mês
- média diária de despesas
- categorias que mais impactam o orçamento
- comportamento financeiro

---

### 🔔 Alertas Inteligentes

Sistema de lembretes inspirado em bancos:

- contas próximas do vencimento
- parcelas
- assinaturas
- pagamentos recorrentes
- alertas financeiros úteis

---

### 🔒 Modo Privado

Proteção visual dos dados financeiros:

- ocultar valores rapidamente
- experiência inspirada em apps bancários
- privacidade instantânea

---

### 📅 Planejamento Financeiro

- visualizar meses passados
- acompanhar o mês atual
- adicionar transações futuras
- organização financeira contínua

---

### 🤖 Atlas Copilot *(Em evolução)*

Assistente financeiro inteligente do Atlas.

Objetivo:

Transformar dados financeiros em insights úteis e recomendações práticas para melhorar sua organização financeira.

## 🚀 Instalação

Clone o repositório:

```bash
git clone https://github.com/thgsz/atlas-control-finance.git
```

Entre na pasta:

```bash
cd atlas-control-finance
```

Instale as dependências:

```bash
npm install
```

Inicie o projeto:

```bash
npm run dev
```
